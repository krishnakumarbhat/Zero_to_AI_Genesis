The Roadmap

A structured journey through the entire landscape of AI and ML, from NumPy-first implementations to cutting-edge LLM alignment.

Mathematics

Key Equations

RL Core (Season 6)

Expected ReturnG_t = sum(gamma^k * R_{t+k+1})
Bellman OptimalityV*(s) = max_a sum P(s',r|s,a)[r + gamma*V*(s')]
PPO Clipped SurrogateL_CLIP = E[min(r_t*A_t, clip(r_t,1-e,1+e)*A_t)]

LLM Alignment (Season 7)

DPO LossL_DPO = -E[log sigma(beta*log(pi/pi_ref)(y_w) - beta*log(pi/pi_ref)(y_l))]
LoRAW = W_0 + delta_W = W_0 + BA, r << d,k
GRPO AdvantageA_i = (r_i - mean(r_1..r_G)) / std(r_1..r_G)
